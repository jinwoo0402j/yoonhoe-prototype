# 윤회_무명 — 이동 조작감 프로토타입 & 레벨 디자인 툴

**윤회_무명** 프로젝트의 핵심 이동 조작감을 프로토타이핑하고, 인게임 레벨 에디터로 레벨을 설계할 수 있는 HTML/JS 기반 툴입니다.

## 🎮 주요 기능

### 이동 시스템 (Hollow Knight 기반)
- 즉시 반응 이동 (가감속 없음)
- 가변 높이 점프 + 비대칭 중력
- 대시 (무적 프레임)
- 벽타기 / 벽점프
- HK 스타일 카메라 (데드존, 지면 포커스, 선행, 룩다운/업)

### 레벨 디자인 툴
- `Tab`으로 플레이 ↔ 에디트 즉시 전환
- 플랫폼 / 벽 / 지면 / 원웨이 / 가시 배치
- 그리드 스냅 (8/16/32px)
- 점프 궤적 오버레이 (풀점프/숏점프/대시 범위)
- Undo/Redo
- JSON 저장/불러오기/클립보드 복사
- 오브젝트 선택, 이동, 크기 조절

## 🚀 사용법
`index.html`을 브라우저에서 열기만 하면 됩니다.

## 조작법
| 키 | 동작 |
|---|---|
| ← → | 이동 |
| Space | 점프 (길게 = 높게) |
| Z / Shift | 대시 |
| Tab | 플레이 ↔ 에디트 |
| H | 도움말 |
